Pokédex Tracker

A tool for tracking your Living Dex progress. Easily toggle between and track your captured Pokémon, find the locations of those left to be captured, and share a public link with others to see how you can help each other out.

This project is open source, and you can find the code on Github (API & website). Feel free to report issues, suggest features, or even submit a pull request!
